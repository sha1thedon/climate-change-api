# climate-change-api
Here is an example of an API that collects all news articles to do with climate change I coded this from watching a youtube tutorial on creating APIS

## To run this project

You can run this project on your local machine. Just pull it down and do the following:

### `npm i`

1. Run the command above to install all the packages

### Start the Project

2. Now run the server:

```bash
npm run start
```

3. Open [http://localhost:8000/news](http://localhost:8000/news) with your browser to see the results of the scrape in your terminal. You can also used certain parameters to get news from individual sources.
